**SIL Open Font License, Version 1.1** (OFL). The OFL permits bundling and redistribution
inside applications.

|---|---|---|---|

| `fredoka_variable.ttf` | Fredoka (variable, `wdth,wght`) | © The Fredoka Project Authors | https://github.com/google/fonts/tree/main/ofl/fredoka |

Full license text: https://openfontlicense.org/open-font-license-official-text/

Each family's original `OFL.txt` ships alongside the font at the source links above.
